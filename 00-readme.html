Readme

The style sheets are built using the following technology:

  • Foundation 4.3.2
  • SASS + Compass
  • jQuery
  • Bundler
  • Grunt
Bundler

Bundler manages any gem dependencies and version that may be required. Run bundle install to fetch any required gems.

Grunt

This version of the epub framework has incorporated Grunt into the development workflow. It now handles SASS compilation and livereload during the development process as well as minification of CSS and Javascript for deployment.

Development

If changes need to be made to the CSS:

  • Grunt CLI, Node.js and Node package manager need to be installed. Please read the online documentation to get started
  • Run npm install to get the require packages
  • Run grunt

Grunt will start watching the working directory for changes

Deployment

To deploy the final build files for the epub

  • Run grunt deploy

This will generate a directory called build with all the necessary files.

  • Provides CSS and javscript concatenation and minification
  • Optimises all graphic file sizes
  • Removes unecessary scripts from the html
  • Copies only the required html, css and javascripts libraries

ePub Directory Structure

Sample pages

Numbered html files are template pages of ePub content

  • 01-section-cover.html
  • 04-sample-content.html
Base elements

Common base styles that are used across all epub titles. base-elements.html

Fonts folder

Icon glyphs / font used across publications

Images folder

Images for inline page content

Images > UI folder

Graphic elements that are used in the theme. Custom panels components + backgrounds, icons etc.

SASS

Contains all the SASS partials for CSS compliation. These are not required for final deployment

CSS

Contains compressed version of the CSS stylesheet

Javascripts

Contains the Foundation framework, jQuery, modernizer and Cambridge files.

Gemfile + Gem.lock

The gem file contains project gem dependencies and versions.
